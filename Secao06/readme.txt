* Novas tag´s de Input
    - for => em label é utilizado com o mesmo id do input para ao clicar no label o input fica selecionado.
    - type(email) => aguarda que o conteúdo do input seja email.
    - type(number) => para utilizar o componente para só aceitar numero e limitar o min e o maximo.
    - type(url) => só aceita url para o conteúdo do campo.
    - placeholder => um dica do que precisa digitar no campo (colocando um texto no campo).
    - type(search) => utilizado para pesquisar algo.
    - type(range) => botão para arrastar de um lado para outro com min e maximo.
    - type(date) => para utilizar para data.
    - type(color) => para modificar a cor.
    - required => utilizado para definir como um campo obrigatório.
    - autofocus => para um campo ficar marcado com focus.

* criar atributos customizados
    - data => utilize o atributo data-<nome> (ex: data-idade, data-altura), para recuperar dados utilizando
        javascript.
