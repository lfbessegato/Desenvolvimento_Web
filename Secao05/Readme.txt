margin => Para colocar espaçamento externo.
    * left
    * right
    * top
    * Bottom 

float => Fica flutuando pela página, o seu fluxo pode ficar lado a lado.
    * right
    * left

Position => Posicionamento.
    * Relativo => alinha a div relativamente a ela mesma, ou movimentação do elemento.
    * Absoluto => Deixa de pertencer ao elemento principal que fez a chamada, absoluta ao parent mais 
    próximo.
    * Fixed => Posição fixa, fica posicionada referente ao view port.

* Elmentos
    * block => Ficam no espaço todo. (h1, p, table)
    * inLine => Ficam em linha, definido pelo seu conteudo (link, span, img)
    * inline block => tamanho definido pela sua largura, mas ficam um embaixo do outro. (display:inline-block)

* links
    <a href="" /> => url de um link
        a:link (links não visitados)
        a:visited (link visitados)
        a:hover (qdo passa o mouse por cima do link)
        a:active (link ativos, qdo clicados)
        text_decoration: none (tira o underline em todos os estados do link)

* css externo 
    <link rel="stylesheet" type="text/css" href="estilo.css" />
