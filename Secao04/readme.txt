CSS 
    * Estilos 
        Inline => Formatação direta no elemento.
        Seletor => Formatação interna a partir da declaração em <body>, escrevendo sobre a tag.
        Classes e IDs => define na tag o nome da classe ou do Id, para distinguir classe de id, faça da 
        seguinte forma: 
            classes = .nome_classe em vários elementos 
            Ids = #nome_id - identificador unico dentro da página (javascript), aplica em um unico elemento.
        
        <div> </div> => Para dividir os elementos / blocks dos assuntos no site a lina.
        <span></span> => Elemento inLine, fica na mesma linha.

        * Border => Para fazer as bordas de um conteúdo (ex. div), e os parâmetros são
            . pixels = 10px; topo, lateral direita, inferior, lateral esquerda (relógio)
            . tipo da borda = solid;
            . cor da borda
        
        * fontes e Cores
            - font-size: tamanho da fonte (pixel);
            - color: cor #BD3909;
            - font-family: tipo de fonte que será utilizada, "Trebuchet MS", Helvetica, sans-serif;

        * formatacao-textos
            .em => medida proporcional a tag pai.
            .% => medida porcentagem de aumento da fonte.
            .px => pixels.
            .font-style => Estilo da fonte.
            .font-weight => Altura da fonte.
            .text-decoration: underline, overline, line-through;
